Tool Reference
Tools tab
JSON
JSON → JSON5
Converts strict JSON to JSON5 — adds trailing commas, allows unquoted keys and comments.
IN JSON  →  OUT JSON5
JSON5 → JSON
Parses JSON5 (with comments, trailing commas, etc.) and converts it to strict JSON.
IN JSON5  →  OUT JSON
Parse Graylog
Extracts and reformats Graylog log lines into a clean, structured JSON object.
IN Graylog log text  →  OUT JSON
Analyze JSON
Inspects a JSON value and reports its structure: key count, nesting depth, value types, and array lengths.
IN JSON  →  OUT Analysis table
JSON Diff uses both panes
Compares two JSON objects and highlights keys that are added, removed, or changed.
IN JSON (left) + JSON (right)  →  OUT Diff in right pane
JSON Path Explorer
Evaluates a JSONPath expression against a JSON document and returns all matching values.
IN JSON + JSONPath expression  →  OUT Matched values
Key Transform
Flat → Wrapped
Wraps every value in {v: …} — the keyed-value format used by some data stores.
IN Flat JSON  →  OUT {key: {v: value}}
Wrapped → Flat
Unwraps {v: …} or {value: …} structures back to plain key-value pairs.
IN Wrapped JSON  →  OUT Flat JSON
Rewrap
Swaps the inner key name between v and value throughout the object.
IN Wrapped JSON  →  OUT Rewrapped JSON
Text
Strip Newlines
Removes all line breaks, producing a single continuous string.
IN Multi-line text  →  OUT Single-line string
Newlines → \n
Converts real newline characters to literal \n escape sequences.
IN Multi-line text  →  OUT Escaped string
\n → Newlines
Converts literal \n escape sequences back to real line breaks.
IN Escaped string  →  OUT Multi-line text
Quote / Unquote / Toggle Quotes
Quote wraps text in double quotes (escaping inner quotes). Unquote strips them. Toggle switches between single and double.
IN Text or quoted string  →  OUT Modified string
Parse Stringified JSON
Parses a JSON value that has been double-encoded (stringified inside a string). Shows the error location if parsing fails, on both the original and unescaped strings.
IN Stringified JSON  →  OUT Parsed JSON or annotated error
Base64 Encode / Decode
Encode converts plain text to a Base64 string. Decode reverses it.
IN Text or Base64  →  OUT Base64 or plain text
JWT Decode
Splits and decodes a JWT's header and payload. Does not verify the signature.
IN JWT token  →  OUT JSON with header + payload + signature
JWT Encode
Signs a JSON payload as a JWT using HMAC (HS256/384/512). Enter the secret and algorithm in the controls below the toolbar. Output is a signed token you can verify at jwt.io.
IN JSON payload + secret + algorithm  →  OUT Signed JWT
HTML Encode / Decode
Encode converts special characters (< > & " ') to HTML entities. Decode reverses it.
IN Plain text or HTML entities  →  OUT HTML entities or plain text
URL Encode / Decode
Encode converts a string to percent-encoded form (e.g. /**/ → %2F%2A%2A%2F). Decode reverses it.
IN Plain text or percent-encoded string  →  OUT Percent-encoded or decoded text
Markdown Preview
Renders Markdown to a styled HTML preview in the right pane in real time when you click Apply.
IN Markdown text  →  OUT Rendered HTML preview
Regex
Regex Tester
Tests a regex pattern against text. Enter the pattern and optional flags in the controls that appear below the toolbar. Reports each match with its index and captured groups.
IN Text + pattern + flags  →  OUT Match list
Data
CSV → JSON
Converts CSV to a JSON array of objects, using the first row as column headers.
IN CSV  →  OUT JSON array
JSON → CSV
Converts a JSON array of objects to CSV with a header row derived from the object keys.
IN JSON array  →  OUT CSV
YAML
YAML → JSON
Parses YAML and converts it to formatted JSON.
IN YAML  →  OUT JSON
JSON → YAML
Converts JSON to YAML format.
IN JSON  →  OUT YAML
Schema
Generate JSON Schema
Infers a JSON Schema by inspecting the structure and types of a sample JSON object.
IN JSON object  →  OUT JSON Schema
Generate Form Schema
Produces field-level form schema definitions (label, type, required) from a JSON object's shape.
IN JSON object  →  OUT Form schema JSON
Generate DAL Map
Builds a DAL (Data Access Layer) subscription map from the shape of a JSON object.
IN JSON object  →  OUT DAL map JSON
Flatten Nested Schema
Collapses nested $defs / definitions references into a single-level schema.
IN Nested JSON Schema  →  OUT Flat JSON Schema
Merge Schemas uses both panes
Deep-merges two JSON Schemas, combining properties and required arrays.
IN Schema (left) + schema (right)  →  OUT Merged schema
Schema Diff uses both panes
Shows fields that were added, removed, or changed between two JSON Schemas.
IN Schema (left) + schema (right)  →  OUT Diff report
Generate TypeScript Interfaces
Produces TypeScript interface declarations from a JSON Schema, including nested types.
IN JSON Schema  →  OUT TypeScript .ts
Generate OpenAPI Schema
Wraps a JSON Schema as an OpenAPI 3.0 components/schemas entry.
IN JSON Schema  →  OUT OpenAPI YAML
Generate Mock Data
Creates realistic sample JSON data that matches the schema's field names and types.
IN JSON Schema  →  OUT JSON mock object
Generate CSV Template
Outputs a CSV header row from the schema's top-level property names.
IN JSON Schema  →  OUT CSV header row
Simplify Schema
Strips verbose metadata ($schema, $id, descriptions, examples) for a leaner schema.
IN JSON Schema  →  OUT Simplified JSON Schema
Generate Table Component
Produces a column-definition config for a data table component, derived from schema properties.
IN JSON Schema  →  OUT Table component config
Validate Object uses both panes
Validates a JSON object against a JSON Schema. Reports pass/fail and lists all validation errors.
IN JSON object (left) + JSON Schema (right)  →  OUT Validation result
Generate Object from Schema
Builds a minimal skeleton JSON object from a schema's required properties. Result goes to the left pane.
IN JSON Schema (right pane)  →  OUT JSON object (left pane)
Utils tab
Time
Timestamp Converter
Converts Unix timestamps (seconds or milliseconds) to human-readable dates across multiple timezones. Click Now to paste the current time. Change the timezone or format to instantly re-render.
IN Unix timestamp  →  OUT Formatted date in selected timezone
Cron Parser
Parses a 5-field cron expression (min hr dom mon dow) into a plain-English description and lists the next 10 scheduled fire times.
IN Cron expression  →  OUT Description + next 10 fire times
Generate
UUID Generator
Generates a cryptographically random UUID v4. Click either format to copy it. Click New UUID to generate another.
OUT UUID with dashes & without dashes
Token Generator
Generates a cryptographically random token of a configurable length. Choose which character sets to include: lowercase, uppercase, numbers, and/or symbols.
OUT Random token string
Convert
Temperature
Converts between Celsius and Fahrenheit. Enter a value in either field and press the arrow button pointing toward the target unit.
IN °C or °F  →  OUT °F or °C
Color Converter
Converts colors between HEX, RGB, and HSL formats. Enter a value in any field and the other two update instantly, along with a live color swatch.
IN HEX, RGB, or HSL  →  OUT The other two formats + swatch
Base Converter
Converts numbers between decimal, binary, octal, and hexadecimal. Edit any field to update the rest.
IN Any base  →  OUT All four bases
Web
URL Encoder / Decoder
Encodes special characters for safe use in a URL, or decodes an encoded URL back to readable text.
IN Plain text or encoded URL  →  OUT Encoded or decoded string
URL Builder
Assembles a complete URL from its parts: protocol, host, port, path, fragment, and query parameters (one key=value per line).
IN URL parts  →  OUT Complete URL
